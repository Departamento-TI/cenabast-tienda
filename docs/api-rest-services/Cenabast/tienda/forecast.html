Demanda Programada

warning

De la respuesta de esta API, sólo se deben considerar los productos que comienza con 10000*

Request​

GET '{{host}}}}/interoperabilidad/tienda/api/v1/materiales/demandasprogramadas/{{rut-solicitante}}/{{rut-destinatario}}/{{mes}}/{{año}}' \
--header 'Authorization: Bearer {{token}}'
