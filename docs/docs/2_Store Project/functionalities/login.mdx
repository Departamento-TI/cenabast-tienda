# Piloto

:::tip
Login with Clave Unica should be implemented through a Single Sign On solution. In this case, Keycloak.

👉 [Kecloak Setup](../../infrastructure/applications/Keycloak)\
👉 [Clave Unica](../../infrastructure/clave_unica)
:::



```mermaid
sequenceDiagram
     Note left of Tienda Cenabast: Usuario ingresa<br>directamente a<br> la nueva tienda de<br>Cenabast
     Tienda Cenabast ->> Keycloak: Usuario inicia login
     Note left of Clave Unica: Usuario es<br>derivado al portal de<br>Clave Unica.
     Keycloak ->> Clave Unica: Redirige a Clave Unica
     Clave Unica -->> Keycloak: Autenticación Correcta con RUT
     Keycloak -->> Tienda Cenabast: Redirigido a la Tienda correctamente.
     Tienda Cenabast ->> SAP Cenabast: Organismos relacionados
     Note left of Tienda Cenabast: Se preguntan los<br>permisos y relaciones<br>del RUT usuario con API.
     SAP Cenabast -->> Tienda Cenabast: Organismos A,B,C.
     Note left of SAP Cenabast: Respuesta con<br>Organismos, líneas<br>de Negocio, y restric-<br>cciones del usuario.
```
